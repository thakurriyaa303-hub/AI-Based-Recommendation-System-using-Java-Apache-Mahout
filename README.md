# AI-Based Recommendation System using Java & Apache Mahout## Internship Task 4This project implements a recommendation system usingApache Mahout to suggest products based on user preferences.## Features- Collaborative filtering- User-based recommendation- Uses Pearson similarity- Works with sample preference data## Technologies Used- Java- Apache Mahout- CSV-based dataset## DatasetThe dataset contains user-item preference ratings usedto generate recommendations.## How It Works1. Loads user preference data2. Finds similar users3. Recommends items based on similarity## AuthorRiya Thakur
